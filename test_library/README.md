# 工业测试图库

每类放置 10 张测试图片（JPG/PNG/WebP），用于功能验证与 Demo 展示。

## 目录结构

| 目录 | 类别 | 适合特征 | 推荐阵列 |
|------|------|----------|----------|
| `crane/` | 起重机/吊机 | 吊臂、斜撑、钢架、滑轮 | 错位砖墙、普通平铺 |
| `factory/` | 厂房与设备 | 屋顶线、门窗、管道、烟囱 | 镜像平铺、普通平铺 |
| `skyscraper/` | 高楼大厦 | 立面、窗格、竖向线条 | 镜像平铺、旋转万花筒 |
| `bridge/` | 桥梁工程 | 弧线、拉索、桁架 | 错位砖墙、普通平铺 |

## 命名规范

建议使用 `{category}_{序号}.jpg` 格式，例如 `crane_01.jpg`。

## 获取测试素材

可从以下来源采集工业建造主题照片：
-  Wikimedia Commons（搜索 crane, factory, bridge, skyscraper construction）
-  自行拍摄的工地/工业设备照片

将图片放入对应子目录后，重启后端服务即可在前端测试图库面板中选取。
