01

项目定位

CupForge（筑纹杯） 是一款面向 工业建造文化 的 纹样生成与 3D 杯具定制 在线工具：用户上传工业建筑照片，系统自动识别主题、 提取视觉特征，生成可编辑纹样，并映射到可 3D 预览、可导出的杯型模型上。

目标用户
文创设计者 · 3D 打印玩家
工业设计爱好者、工业文化展览/课程参与者
核心场景
照片 → 图案 → 3D 杯具
厂房、大桥、塔吊等工业意象转化为杯身纹样
交付形态
浏览器实时预览 + STL 导出
支持 3D 打印或进一步建模

设计原则

  • 文化主题明确 — 聚焦工业建筑视觉语言，非通用图片编辑器
  • 算法 + 交互结合 — 后端 CV 特征提取，前端 Canvas / Three.js 实时编辑
  • 双工艺路径 — 贴图流程（平面纹样）与浮雕流程（几何位移、可打印）并行
  • 低门槛 Web 化 — 无需安装，部署后一个链接即可使用

技术栈

层级技术选型职责
后端Python · OpenCV · scikit-learn · FastAPI · trimesh预处理、主题识别、视觉形态、光滑杯型导出
前端React · TypeScript · Canvas 2D · Three.js纹样阵列、实时预览、3D 交互、浮雕 STL 导出
部署Docker · GitHub · Render单容器托管前后端，公网访问

后端图像处理管道

入口：POST /api/upload → preprocess.run_pipeline()

步骤模块算法
1load_and_resize最长边 ≤ 1024px
2denoise_and_enhance双边滤波 + CLAHE
3detect_edgesCanny（阈值可调）
4clean_contours过滤小面积轮廓
5extract_paletteKMeans 5 色，过滤高亮背景
6expressions5 种视觉形态生成
7industrial_recognize启发式多特征打分

07

技术亮点

  • 主题化而非通用化 工业建筑识别 + 工业意象关键词贯穿方案生成与 UI 叙事
  • 前后端分工清晰 重 CV 计算放后端，重交互与 3D 放前端；浮雕 STL 纯前端生成减轻服务器压力
  • 算法可配置 config.py 集中管理阈值，前后端通过 /api/config 同步枚举
  • 双工艺闭环 同一套取样/阵列能力服务贴图预览与浮雕几何两种交付
  • Web 原生交付 Docker 一键部署，Render 免费 tier 可分享链接演示

已知限制 · Q&A 备用

限制说明
会话内存存储服务重启或冷启动后会话丢失，需重新上传
贴图 STL 不含纹样几何贴图流程导出仅为光滑杯壳；纹样为渲染贴图
免费部署性能Render 512MB + 冷启动，大图上传较慢
无用户账号无登录、无历史作品持久化
单页无 URL 路由刷新页面会回到首页，状态不保留
